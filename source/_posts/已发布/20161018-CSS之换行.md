以下收集整理的是有关于CSS样式文字换行的方法

<!--more-->


#### 一、强制不换行

```css3
div{
white-space:nowrap;
}
```

#### 二、自动换行

```css3
div{ 
word-wrap: break-word; 
word-break: normal; 
}
```

#### 三、强制英文单词断行

```css3
div{
word-break:break-all;
}
```

#### 四、多行显示省略号

```css3
div {
overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 3;
-webkit-box-orient: vertical;
}
```
